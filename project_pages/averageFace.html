The Brief

The Average Face project was about exploring image editing at the scale of image pixels. A set of images was processed with linear interpolation to arrive at the final overall average image. This project was one of four projects submitted as the capstone submission to "CM2030 Graphics Programming".

The experience

The user of this application gets to explore the process of liner interpolation, by selecting the range or amount of interpolation done to the final image.

Building Blocks

This application use's 30 images of faces from the set of images generated entirely by AI to create an average image.

To arrive at the average image, the images are processed through two for loops to sum the pixel values which are then divided by the total number of images. The resulting value is then passed to a linear interpolation function which sets the final pixel value. To allow the application to produce an image that is between the beginning image and the final output, the mouse position is mapped to a value between 0 and 1 which tells the linear interpolation function how much to interpolate.

User Interaction

The user of this application changes the interpolation amount by placing the mouse within the blue rectangle at the bottom of the application. The left side of the rectangle is 0% interpolation and the right side is 100%

Conclusion

This application was a good example at the processes required for image editing and introduced the idea of using code to create new images procedurally. This project was the second of four worked on as the capstone submission for the mandatory course Graphics Programming.
