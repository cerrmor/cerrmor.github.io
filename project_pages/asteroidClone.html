The Brief

The goal of this application was to explore the concept of applying physical properties such as gravity to an animation. By implementing simple physics functions and applying them to elements within the game, a basic physics engine was created. This allowed objects within the game to respond more realistically to influences from the application such as the lack of friction within space. This creates for a more enjoyable playing experience as it adds to the illusions built by the game. This project was one part of four that served as the midterm submission for CM2030 Graphics Programming.

The experience

The goal of the Astroids clone is to survive as long as possiable. The user controles a ship and tries to destroy all the falling astroids before they hit the planet. The user controls the ship by pressing the directional arrow keys and fires the weapon systems by pressing spacebar. If the ship touches an astroid or the planet surface, the game ends and must be restarted. Similar to classic astroids when the ship moves in any direction it will continue to move for a time, due to low friction forces. Additionaly, if the users ship approaches the planet surface it will be drawne towards it due to gravity. These physical forces add essential depth to the experience of the game and show why applying physics to an animation can enhance the end product.

Building Blocks

The application was built using the p5.js JavaScript library, and some basic HTML code. There were four main componentes essential for this game to function, a system of collision detecton, bullet and astroid managment, and finally a system for managing the ship. Each was established as its own class element to keep the code clean and modular. The astroid game elements were propelled towards the planet by applying a small gravity force to the astroides acceleration. This all served to create a compelling game experienc.

User Interaction

The user interactes with the experience by controlling the ships direction with the arrow keys and fireing its gun system to destroy asteroids with the spacebar

Conclusion

This application was effective at showcasing the value of applying physical forces to game elements and animations. furthermor, By doing so, it created a better game experience that was more fun to play.
